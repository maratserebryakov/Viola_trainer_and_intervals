🌸🎻 Альтушка-тренажёр
======================

Милейший интерактивный тренажёр для альта прямо в браузере.  
Позиции, пальцы, ноты, интервалы — всё, чтобы подружиться с грифом и чувствовать себя уверенно.

🌷🌿 Что внутри?
-----------------

✨ **Три основных режима:**
- `1️⃣` струна + позиция + палец → назови ноту  
- `2️⃣` струна + нота → определи позицию и палец  
- `🎲` случайное чередование вопросов  

🌸 **Низкие и высокие пальцы:**
Тренируйся с «нормальным» положением и добавляй полутоны туда-сюда — как в настоящей музыке.

🌼 **Королевский блок интервалов:**
- `Инт. 4️⃣` — дана нота и интервал → построй следующую  
- `Инт. 5️⃣` — даны две ноты → назови интервал между ними  
- `Инт. 6️⃣` — случайный интервал вверх или вниз  

💐 **Альтовый строй:**
Струны `A – D – G – C`, открытые ноты подписаны по‑русски (ля мал, ре мал, соль бол, до бол).

🌺 **Красивый интерфейс:**
Пастельные тона, округлые карточки, цветочное настроение и никакой скуки.

🌻 Как запустить?
------------------

Просто открой файл `index.html` в любом современном браузере —  
ни сервер, ни установка не нужны.

🌹 Примеры ответов
-------------------

- `ля#мал` — ля-диез малой октавы  
- `сиbмал` — си-бемоль малой октавы  
- `мивтор` — ми второй октавы  
- `ч5`, `б3`, `м2`, `ув4` — для интервалов  

🌾 Октавы:
`бол` → `мал` → `перв` → `втор` → `трет`
